# ABK Scientific — AI agent entry point

Before working on this repo, read:

- **`dev_docs/AGENTS.md`** — stack, commands, directory map, conventions, and hard rules.
- **`dev_docs/PROJECT-CONTEXT.md`** — business goal, positioning, decision log, and roadmap.

All marketing copy lives as typed data in `src/content/*.ts` — edit copy there, not in JSX.
Run `npm run build` before any deploy/PR. Never fabricate clients, metrics, or testimonials.
